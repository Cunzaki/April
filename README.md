# April

Modular cheat script for **Fallen Survival** on [Project Vector](https://project-vector-1.gitbook.io/vector-lua-engine/).

## Requirements

- Project Vector with Vector Lua Engine
- Fallen Survival loaded in-game

## How to load (important)

Vector only shows script menu options when menu registration runs in the **file you execute**. Remote `load_url` / `loadstring` loaders create an empty April tab with no options — that is expected.

### Recommended: `april_bundle.lua`

1. In Vector, click **Open Folder** in the script editor (opens your Scripts directory).
2. Copy the entire `April` folder from this repo into that Scripts directory.
3. **Load Script** → pick `April/april_bundle.lua` → **Execute Script**.

You should see in the console:

```
[April] bundle starting...
[April] Menu groups registered: 13
[April v3] Loaded — 3.0.0
```

All options appear under the **April** tab in the Scripts panel (Aimbot, Recoil, Players, World, etc.).

### Alternative: modular dev loader

If you are editing individual files in `core/`, `features/`, etc.:

1. Copy the `April` folder into Vector Scripts (same as above).
2. **Load Script** → `April/loader.lua` → **Execute**.

Regenerate the bundle after module changes:

```bash
node tools/bundle_april.mjs
```

### Paste fallback

Open `April/april_bundle.lua`, copy the entire file into your Vector script, and **Execute Script**.

## What does NOT work

Do **not** use a bare remote one-liner:

```lua
utility.load_url("https://raw.githubusercontent.com/.../loader_remote.lua")  -- menu stays empty
```

Menu items registered inside `load_url` / `loadstring` chunks are not bound to the Scripts UI.

## Config files

Settings save to:

`%LOCALAPPDATA%\Project Vector\Scripts\April_Slot_1.txt`

Use **Config** group in the April menu to save/load slot 1.

## Updating

```bash
cd Scripts/April
git pull
node ../../tools/bundle_april.mjs   # if you have the dev repo; or re-copy april_bundle.lua
```

Then unload the old script and execute `april_bundle.lua` again.

## Development

Source modules live in `core/`, `game/`, `features/`, `menu/`. The bundle is generated by `tools/bundle_april.mjs` in the parent dev repo.
